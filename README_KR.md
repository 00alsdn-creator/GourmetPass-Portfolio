# GourmetPass (포트폴리오 버전)

(Spring MVC 기반 팀 프로젝트)

## 📌 프로젝트 개요

GourmetPass는 맛집 예약 및 대기 시스템을 목표로 한 웹 애플리케이션입니다.

## 👨‍💻 담당 기능 (핵심)

- 이미지 업로드 및 압축 처리
- Ajax 비동기 업로드
- 매장 상세 페이지 UI
- 즐겨찾기 기능

## 🛠 사용 기술

- Java
- Spring MVC
- MyBatis
- Oracle DB
- JavaScript / Ajax
- JSP / JSTL

## ⚙ 개발 환경

- JDK 11
- STS
- Oracle SQL
- Git / GitHub
