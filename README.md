# GourmetPass

飲食店予約・待機管理システム  
(Spring MVC 기반 チームプロジェクト)

---

## 📌 プロジェクト概要

GourmetPassは、飲食店の予約・待機管理を目的としたWebアプリケーションです。  
店舗情報、画像管理、お気に入り機能などを通して、ユーザーが店舗を探しやすいサービスを目指しました。

---

## 🧑‍💻 担当機能（重点）

本プロジェクトでは主に以下の機能を担当しました。

### ■ 店舗画像管理機能
- 画像アップロード機能の実装
- クライアント側での画像圧縮処理
- Ajaxを利用した非同期アップロード
- サーバー側での画像再最適化処理
- サムネイル表示対応

### ■ 店舗詳細ページUI
- JavaScriptスライダー実装
- 表示画像の制御（表示 / 非表示）

### ■ お気に入り機能
- ユーザー別お気に入り登録
- 重複登録防止ロジック実装
